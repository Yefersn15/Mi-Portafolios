Ejercicios Evaluativos

Prácticas de HTML básico: listas, encabezados, enlaces, imágenes, formularios y página de error.

1. Listas ordenadas <ol> · <li> Frutas y verduras en listas numeradas.
2. Encabezados H1–H6 <h1> … <h6> Todos los niveles de encabezado.
3. Enlaces y anclas <a href> · name Enlaces a otras páginas y saltos internos.
4. Imágenes como enlaces <a> + <img> Miniaturas que enlazan a ejercicios.
5. Formulario con JS form · input · select · JavaScript Validación condicional con alertas.
6. Página 404 HTTP 404 Plantilla de error no encontrado.
