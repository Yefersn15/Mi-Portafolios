EncomiExpress

Panel web administrativo para OsvaldoC Mensajería y Logística S.A.S. Gestiona ventas, encomiendas, rutas, conductores, vehículos y anticipos. Incluye modo oscuro, paleta de colores personalizable y dos modos de navegación.

Tecnologías: React 19, Material UI (MUI), Vite, React Router, Context API.
